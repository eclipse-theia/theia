    • redo(): void | Thenable<void>

      Redo the edit operation.

      This is invoked by Theia when the user redoes this edit. To implement redo, your extension should restore the document and editor to the state they were in just after this edit was added to Theia's internal edit stack by CustomEditorProvider.onDidChangeCustomDocument.
