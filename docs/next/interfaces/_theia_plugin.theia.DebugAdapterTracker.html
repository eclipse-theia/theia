    Interface DebugAdapterTracker

    A Debug Adapter Tracker is a means to track the communication between VS Code and a Debug Adapter.

    interface DebugAdapterTracker {
        onDidSendMessage?(message: any): void;

        onWillReceiveMessage?(message: any): void;
