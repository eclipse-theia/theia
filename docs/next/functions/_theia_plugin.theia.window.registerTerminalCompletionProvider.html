    Function registerTerminalCompletionProvider

    • registerTerminalCompletionProvider<T extends TerminalCompletionItem>(
          id: string,
          provider: TerminalCompletionProvider<T>,
          ...triggerCharacters: string[],
      ): Disposable

      Register a completion provider for terminals.

      Type Parameters

      • T extends TerminalCompletionItem

      Parameters

      • id: string

        The unique identifier of the terminal provider, used as a settings key and shown in the information hover of the suggest widget.

      • provider: TerminalCompletionProvider<T>

        The completion provider.

      • ...triggerCharacters: string[]

      Returns Disposable

      A Disposable that unregisters this provider when being disposed.

      Example: Register a provider for an extension

      window.registerTerminalCompletionProvider('extension-provider-id', {
       provideTerminalCompletions(terminal, context) {
           return new TerminalCompletionList([
               { label: '--version', replacementIndex: Math.max(0, context.cursorPosition - 2), replacementLength: 2 }
           ]);
       }
      });
