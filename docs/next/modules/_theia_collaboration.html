    Module @theia/collaboration


    ECLIPSE THEIA - COLLABORATION EXTENSION

    Description

    The @theia/collaboration extension features to enable collaboration between multiple peers using Theia. This is built on top of the Open Collaboration Tools (GitHub) project.

    Note that the project is still in a beta phase and can be subject to unexpected breaking changes. This package is therefore in a beta phase as well.
