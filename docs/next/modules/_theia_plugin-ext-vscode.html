    Module @theia/plugin-ext-vscode


    ECLIPSE THEIA - PLUGIN-EXT-VSCODE EXTENSION

    Description

    The @theia/plugin-ext-vscode extension contributes functionality to be able to successfully run VS Code extensions in the application. The extension is mandatory for any application which requires VS Code extension support.
