    Module @theia/ai-google


    ECLIPSE THEIA - AI GOOGLE EXTENSION

    Description

    The @theia/ai-google integrates Google's models with Theia AI. The Google API key and the models to use can be configured via preferences. Alternatively the API key can also be handed in via the GOOGLE_API_KEY environment variable.
